### **Tech Stack:**

- Nextjs
- TypeScript
- Dexie.js
- Tailwind

### **Setting up locally**

```bash
git clone https://github.com/ronitrajfr/betterwrite.git
cd betterwrite
npm install
```

And then run:

```bash
bun run dev
```
